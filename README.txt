Tungle (http://tungle.com) is a system for helping you manage your 
calendars and assist in scheduling meetings.

This module simply integrates the Tungle.me widget onto the user profile 
page based on email address.

Future
------

Potential future features / functions.

* show calendar on a tab of the user profile page
* have a "book meeting with" link / workflow
* make a Feature that integrates with Open Atrium
* show availability of all members in a group / selected members
* Tungle CCK field
